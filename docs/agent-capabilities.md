# יכולות הסוכן — מסמך מקיף

## 1. יכולות הסוכן

### 📧 Gmail
| יכולת | פירוט |
|-------|-------|
| קריאת מיילים | מיילים לא נקראים עם סינון חכם (ללא ניוזלטרים) |
| חיפוש | לפי שולח, נושא, מילות מפתח, טווח תאריכים |
| שליחה | דורשת אישור מפורש לפני שליחה |
| חוקים אוטומטיים | סיווג מיילים לפי שולח/נושא, העברה לתיקייה, סימון כנקרא |

### 📅 Google Calendar
| יכולת | פירוט |
|-------|-------|
| צפייה | אירועי היום / מחר / 7 ימים קרובים (כל היומנים) |
| יצירה | אירוע עם כותרת, תאריך, שעות, מיקום, תיאור, יומן ספציפי |
| עריכה | שינוי כל שדה של אירוע קיים |
| מחיקה | מחיקת אירוע לפי ID |
| העברה | העברת אירוע בין יומנים |

> **חשוב:** כל פעולת כתיבה (יצירה/עריכה/מחיקה) דורשת אישור מפורש ("כן") לפני ביצוע.

### ⏰ תזכורות
- קביעת תזכורת בשפה חופשית: "מחר ב-17:00", "בעוד שעה", "ביום שישי"
- המרה אוטומטית לתאריך ISO מדויק לפי שעון ישראל
- שליחה אוטומטית ב-WhatsApp בזמן שנקבע

### 🔍 חיפוש אינטרנטי
- חיפוש דרך Tavily עם 5 תוצאות וסיכום
- **בקבוצות:** דורש אישור Owner לפני ביצוע

### 🧠 זיכרון לטווח ארוך
- שמירת עובדות: אנשים, פרויקטים, העדפות, אירועים
- קטגוריות: `people`, `projects`, `preferences`, `episodic`
- שליפה לפי מילות מפתח
- שמירה אוטומטית כשהמשתמש מזכיר שם/פרויקט/העדפה

### 👥 אנשי קשר וקבוצות
- `save_contact` — שמירת איש קשר (שם, טלפון, מייל, הערות)
- `get_contact` — חיפוש לפי שם או טלפון
- `save_group` — שמירת קבוצת WhatsApp/טלגרם
- הוספה ידנית: "הוסף איש קשר: [שם], [טלפון], [מייל]"
- שאילת מייל: כשחסר מייל לשליחה → "מה המייל של [שם]?"

### ⚙️ העדפות מערכת
- שמירה אוטומטית של העדפות קבועות ב-`system_notes`
- לדוגמה: "לא זמין בימי שישי אחרי 14:00"

### 🌅 סיכום בוקר
- נשלח ב-07:30 (ניתן להגדרה) כ-voice note
- כולל: אירועי היום, מיילים ממתינים, תזכורות פתוחות

## 3. שימוש בקבוצות

### Owner בקבוצה
- גישה מלאה לכל המידע
- יכול לשאול כל שאלה בלי @mention בטלגרם
- ב-WhatsApp: צריך @mention

### חבר מאושר (approved)
- **מותר:** "האם הבעלים פנוי ב-X?" → רק פנוי/לא פנוי
- **אסור:** פרטי אירוע, מיילים, תזכורות, זיכרון אישי
- חיפוש אינטרנטי: דורש אישור Owner

### חבר לא רשום
1. מסר ראשון → בקשת מייל
2. שלח מייל → pending_approval
3. Owner אישר → approved

### רישום חבר חדש (awaiting_intro)
כשהבוט מצטרף לקבוצה חדשה:
1. שולף רשימת חברים
2. לחברים לא מוכרים → `status=awaiting_intro`
3. שולח הודעת ברוכים הבאים + בקשת שם ומייל
4. חבר שולח שם + מייל → `pending_approval` → Owner מאשר

### אסור תמיד בקבוצה
- מיילים פרטיים של הבעלים
- תזכורות אישיות
- פרטי אירועים (שם, מיקום, משתתפים)

## 4. הגדרות בטיחות

### זיהוי Owner
- **WhatsApp:** השוואת last-9 ספרות של מספר הטלפון → `OWNER_PHONE`
- **Telegram:** השוואת `user_id` → `OWNER_TELEGRAM_ID`
- **Web:** סיסמת כניסה → `WEB_CHAT_PASSWORD`

### הגנות ב-Baileys bridge
| הגנה | פירוט |
|------|-------|
| 401 guard | 2× loggedOut תוך 60 שניות → עצירה מוחלטת + התראה |
| Max reconnect | 10 ניסיונות עם backoff (30s/60s/120s/5min) → halt |
| Warmup 60s | אין עיבוד הודעות ב-60 שניות הראשונות אחרי חיבור |
| Session dedup | `seen` set עם TTL 30 שניות |
| `deviceReady` guard | שליחה חסומה עד `receivedPendingNotifications=true` |

### הגנות ב-backend
| הגנה | פירוט |
|------|-------|
| Injection detection | 20+ regex patterns, zero-width chars, base64 |
| Owner-only DM | DM מ-non-owner נדחה לפני כל עיבוד |
| Approval system | פעולות כתיבה (מייל, יומן) דורשות "כן" מפורש |
| Rate limit | עד 100 קריאות Claude לשעה (ניתן להגדרה) |
| Pending expiry | פעולות pending נמחקות אחרי 3 דקות |
| Security hardening | הרשאות לא ניתנות לשינוי בשיחה |

## 5. רשימת בדיקות

### WhatsApp
- [ ] QR scan מוצלח → `[WHATSAPP] connected! isNewLogin=true`
- [ ] הודעת טקסט מ-Owner → תשובה מהסוכן
- [ ] Voice note מ-Owner → תמלול + תשובה קולית
- [ ] "תכתוב במקום" → תשובה טקסט (לא קול)
- [ ] הודעה מ-non-owner → נדחית שקטה
- [ ] קבוצה: @mention בוט → תשובה בקבוצה
- [ ] קבוצה: חבר לא מוכר → בקשת מייל
- [ ] 401 כפול תוך 60s → `[FATAL]` + עצירה

### Telegram
- [ ] הודעת Owner DM → תשובה
- [ ] Voice note DM → תמלול + תשובה קולית
- [ ] הוספת בוט לקבוצה → הודעת ברוכים הבאים
- [ ] Non-owner בקבוצה בלי @mention → התעלמות
- [ ] Non-owner עם @mention → בקשת רישום
- [ ] Owner בקבוצה ללא @mention → תשובה מלאה

### Gmail
- [ ] "תראה לי מיילים לא נקראים" → רשימת מיילים
- [ ] "שלח מייל ל-X" → בקשת אישור → "כן" → נשלח
- [ ] "כן"/"לא" לאישור → מבוצע/מבוטל

### Google Calendar
- [ ] "מה יש לי היום?" → אירועי היום
- [ ] "צור פגישה מחר ב-10:00" → בקשת אישור → "כן" → נוצר
- [ ] "מחק את פגישת X" → בקשת אישור → "כן" → נמחק

### אנשי קשר
- [ ] "הוסף איש קשר: דוד כהן, 052..., david@..." → נשמר + אישור
- [ ] "מצא את דוד כהן" → פרטים + מייל
- [ ] "שמור את הקבוצה הזו" → שמירה + אישור

### תזכורות
- [ ] "הזכר לי מחר ב-17:00 לקנות חלב" → "✅ תזכורת נקבעה"
- [ ] בזמן שנקבע → קבלת הודעה ב-WhatsApp

### Web Chat
- [ ] כניסה עם סיסמה → הצלחה
- [ ] שליחת הודעה → תשובה
- [ ] לחיצה על 🎤 → הקלטה → תמלול → תשובה
- [ ] "תקריא" → audio player עם waveform

### בדיקות קצה
- [ ] הודעה בזמן warmup 60s → נדחית (מוצג בלוג)
- [ ] Injection attempt בקבוצה → נדחה + התראה לOwner
- [ ] Backend restart → session משוחזר מ-DB
- [ ] Bridge restart → session משוחזר → חיבור ללא QR
